Futoshiki Puzzle Solver

Puzzle solver using genetic algorithms and island models to tackle complex constraint satisfaction problems through evolutionary programming.

Project Overview

What is Futoshiki?

Futoshiki (不等式, "inequality") is a logic puzzle similar to Sudoku. The goal is to fill a grid with numbers such that:

  • Each row contains all numbers from 1 to N exactly once
  • Each column contains all numbers from 1 to N exactly once
  • All inequality constraints between adjacent cells are satisfied

The Challenge

Futoshiki is a constraint satisfaction problem (CSP) that becomes computationally challenging as grid size increases. Traditional backtracking algorithms can be slow for larger puzzles with many constraints.

Evolutionary Approach

This project uses genetic algorithms with an island model to solve Futoshiki puzzles:

Chromosome Representation

Each candidate solution (chromosome) is represented as a 2D array where each row is a permutation of numbers 1 to N. This guarantees row constraints are always satisfied.

Fitness Function

The fitness measures the number of constraint violations:

  • Column uniqueness violations
  • Inequality constraint violations

A perfect solution has fitness 0.

Genetic Operators

  • Selection: Tournament selection with elitism
  • Crossover: Row-wise crossover preserving permutation structure
  • Mutation: Swap mutation within rows

Island Model

Multiple populations evolve independently on separate "islands" with occasional migration of best individuals. This helps maintain diversity and avoid premature convergence.

Results

The evolutionary approach successfully solves most standard Futoshiki puzzles (5x5, 6x6, 7x7) within reasonable time. The island model significantly improved convergence compared to a single-population GA.

Key Learnings

  • Importance of constraint-aware representation
  • Parameter tuning for population size and mutation rates
  • Benefits of parallel evolution with migration
