Particle Escape (Brownian Motion)

Simulation study exploring the escape of a particle from an intracellular region using Brownian motion dynamics and statistical analysis.

Python NumPy pandas Simulation
View Code

Project Overview

Introduction

This project models the stochastic process of a particle escaping from a confined intracellular region, inspired by problems in molecular biology and biophysics. Brownian motion (or Wiener process) is used to simulate the random movement of particles in a fluid environment.

The Physical Problem

In biological systems, molecules must often escape from one compartment to another. Understanding the statistics of escape times is crucial for:

  • Drug delivery and absorption rates
  • Cellular signaling processes
  • Ion channel dynamics
  • Protein folding kinetics

Mathematical Model

The particle's motion is governed by the Langevin equation:

dx = √(2D) · dW(t)

Where D is the diffusion coefficient and W(t) is a standard Wiener process. The boundary conditions define the confined region, and we track when the particle first exits this region.

Simulation Approach

  • Discrete time steps: Using Euler-Maruyama method for SDE integration
  • Boundary handling: Absorbing and reflecting boundary conditions
  • Monte Carlo sampling: Running thousands of trajectories to build escape time distributions

Analysis Performed

  1. Mean First Passage Time (MFPT) calculation
  2. Escape time distribution fitting (exponential, Gamma)
  3. Dependence on diffusion coefficient
  4. Effect of boundary geometry
  5. Comparison with analytical solutions where available

Key Findings

The simulation results align well with theoretical predictions for simple geometries. The project demonstrates how computational methods can provide insights into complex stochastic processes that may not have closed-form solutions.
