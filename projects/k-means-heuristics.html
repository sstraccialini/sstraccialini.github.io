K-Means Clustering Heuristics

Project Overview

Abstract

In this thesis we present some classic heuristics for the construction and improvement of a solution to the Minimum-sum-of-squares Clustering Problem (MSSC). We then propose a modified version of the extended-Hartigan algorithm, and provide an empirical analysis in terms of running time and quality of the solution, comparing different combinations of heuristics on some classic datasets.

The K-Means problem, also known as Minimum-sum-of-squares Clustering (MSSC), is one of the most important clustering problems. Its goal is to partition a dataset of observations into K groups, called clusters, such that each observation is associated with the cluster whose centroid (center of mass, or "mean" of the cluster) is closest to it. The objective is to minimize the total squared distance from each point to the center of its cluster. The general problem is NP-Hard even with K = 2, so heuristics must be used in practice.

Proposed Algorithm: Extended-Hartigan

The algorithm proposed is based on the principle of moving a point from one cluster to another only if the objective function decreases. Unlike the classic Lloyd's algorithm that works in two phases (assignment and update), this method performs incremental updates that can lead to faster convergence and better local optima.

Pseudocode

Algorithm: ext-Hartigan (Initialization, K, max_it)
  i ← 0
  X ← Initialization(K)
  converged ← False
  
  WHILE i < max_it AND NOT converged DO
    converged ← True
    FOR EACH x ∈ X DO
      δ_best = +∞
      h_best = cluster(x)
      
      FOR h ∈ {1,...,K} DO
        δ = cost_of_move(x, cluster(x), h)
        IF δ < δ_best THEN
          δ_best ← δ
          h_best ← h
      
      IF h_best ≠ cluster(x) THEN
        converged ← False
        move_to_cluster(x, h_best)
    i ← i + 1
  
  RETURN X

Experimental Setup

The algorithms were tested on classic clustering datasets:

  • A-Sets (A1-A3): Synthetic datasets with varying cluster sizes
  • Bridge: Real-world dataset with complex structure
  • House: Standard benchmark dataset

Results

The extended-Hartigan algorithm, when combined with intelligent initialization strategies (K-Means++ or random farthest), consistently achieved better objective function values compared to the standard Lloyd's algorithm, while maintaining competitive running times.
