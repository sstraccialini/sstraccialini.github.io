Project Overview

Abstract

SuBERT (Sumerian BERT) is an end-to-end pipeline designed to recognize and translate ancient Sumerian cuneiform tablets. The project combines state-of-the-art computer vision models for character detection with Transformer-based sequence-to-sequence models for translation.

The Challenge

Sumerian is one of humanity's earliest written languages, dating back to approximately 3100 BCE. Translating ancient cuneiform tablets presents unique challenges:

  • Limited dataset availability
  • Complex wedge-shaped characters
  • Damaged or partially preserved tablets
  • Multiple writing systems evolved over millennia

Pipeline Architecture

The SuBERT pipeline consists of three main stages:

1. Character Detection

Using YOLOv5 and Faster R-CNN models to detect and localize individual cuneiform signs within tablet images. The models were trained on the SumTablets dataset, a curated collection of annotated tablet photographs.

2. Character Recognition (OCR)

A custom CNN-based classifier identifies each detected cuneiform sign, mapping it to its Unicode representation or transliteration.

3. Machine Translation

The recognized Sumerian text is then translated to English using fine-tuned Transformer models:

  • BART: Bidirectional encoder for robust understanding
  • GPT-2: Autoregressive generation for fluent output
  • Custom Seq2Seq: Baseline comparison model

Dataset: SumTablets

We curated a dataset of Sumerian tablets with parallel translations, drawing from:

  • CDLI (Cuneiform Digital Library Initiative)
  • ETCSL (Electronic Text Corpus of Sumerian Literature)
  • Museum collections with digital catalogs

Results

The pipeline achieves promising results on held-out test sets, with the BART-based translation model showing the best performance in terms of BLEU score and human evaluation. The project demonstrates the potential of modern NLP techniques for preserving and understanding ancient languages.

Future Work

  • Expand training data with more tablet images
  • Incorporate attention visualization for interpretability
  • Add support for other cuneiform languages (Akkadian, Hittite)
  • Deploy as a web service for researchers
